Here is a list of some WooF-powered sites:

  • Yousuf International This is a traditional searchable e-commerce application. It should be noted that this project has been running reliably without any code changes for the past 3 years.
  • Bernard Hodes Group's custom Exchange tool (not the mail server!) - This is a very sophisticated online tool for Bernard Hodes Group, a recruitment advertising agency, to communicate with their clients regarding advertising placements. This site is password-protected; a demo account is set up with username 'demo user' and password 'demo'. Please feel free to contact us if you would like a walk-through on many of the sophisticated behind-the-scenes features, like dynamic PDF rendering, on the fly historical currency conversion, and using multiple EOModel files to do aggregate searches from many different FileMaker systems.
  • University of Kentucky: Username and password are both 'demo'. This is a system that is used to help teachers in the state of Kentucky deal with learning-disabled students. It was recently switched from FileMaker as a database to MySQL, but the WebObjects programming and the features of the site are unchanged.
  • University of Strasbourg ( Apple did a write-up on this solution which uses WooF: here is the original french version and here is the Google translation to English )
